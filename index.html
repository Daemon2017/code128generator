code128generator

code128generator это генератор штрих-кодов формата Code 128, которые используются в тестовых наборах FTDNA. За один раз можно сгенерировать не более 100 штрих-кодов.
code128generator is a generator of Code 128 barcodes used in FTDNA test kits. You can generate no more than 100 barcodes at a time.
Предупреждение: подготовка ответа может занять до 30 секунд. Загрузка ZIP-файла начнется автоматически.
Warning: it may take up to 30 seconds to prepare a response. The download of the ZIP file will start automatically.
